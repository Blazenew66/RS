## 📝 部署步骤

1. 将代码推送到 GitHub
2. 访问 https://streamlit.io/cloud
3. 连接 GitHub 账户
4. 选择仓库
5. 设置主文件为 `app.py`
6. 点击 Deploy

## ⚠️ 注意事项

- Streamlit Cloud 会自动安装 `requirements.txt` 中的依赖
- 首次部署可能需要几分钟
- 如果遇到问题，查看 Streamlit Cloud 的日志
